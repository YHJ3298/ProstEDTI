What is Drug-Target Interaction Prediction?

Drug-Target Interaction (DTI) prediction is a core link in drug discovery. By using computational methods to predict the binding potential between small-molecule drugs and biological targets (such as proteins), it can significantly reduce experimental costs and shorten development cycles.

The ProstEDTI system innovatively integrates multi-dimensional features with advanced machine learning algorithms to build a prediction model with both high accuracy and interpretability. It not only outputs reliable interaction prediction results but also reveals the key biological mechanisms behind the predictions, providing in-depth references for drug design and optimization.

Core Technology Framework

Four core technology modules work synergistically to achieve dual breakthroughs in prediction accuracy and interpretability

Dual-Representation Fusion Strategy

Integrates 64-dimensional molecular embeddings from Mol2vec and 1024-dimensional protein embeddings from ProstT5, comprehensively capturing drug molecular structure features and target sequence information to lay the foundation for accurate prediction.

ENN-SHAP Collaborative Optimization

Eliminates noisy samples through Ensemble Neighborhood Sampling (ENN) and selects key dimensions based on SHAP feature contribution, effectively improving model generalization ability and computational efficiency.

Ensemble Model Architecture

Constructs an XGBoost-LightGBM ensemble model, combining the ability to capture nonlinear feature interactions with efficient computing performance, balancing prediction accuracy and running speed.

Multi-level Interpretability Verification

Innovatively integrates LIME interpretation technology to achieve visual presentation from global feature importance to local decision logic, making prediction results traceable and justifiable.
